== Description ==

Get Tracking Info on your Package !!  Search your tracking number through FedEx, UPS, DHL, & USPS databases.

It's pretty simple to run, just drop it in the plugin directory and activate.  

Then add it as a widget, ShipmentTracker, to your blog.


== Installation ==

1. Upload and drop the plugin folder, ShipmentTracker, to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress Admin.
4. Insert the ShipmentTracker plugin into your theme by using the sidebar widget titled "ShipmentTracker".
